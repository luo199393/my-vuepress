### http状态码

::: tip 200
服务器请求成功
:::

::: danger 400
服务器不理解请求
:::

::: danger 401
请求身份验证
:::

::: danger 403
服务器拒绝请求
:::

::: danger 404
网页不存在
:::

::: danger 405
禁用请求中的指定方法
:::

::: danger 406
无法使用请求的内容特性响应请求的网页
:::

::: danger 410
请求的资源永久删除后
:::

::: danger 500
服务器错误，无法完成请求
:::

::: danger 501
服务器不具备完成请求的功能
:::

::: danger 502
服务器作为网关或者代理，上游服务器收到无效请求
:::

::: danger 503
服务器目前无法使用，通常是暂时状态
:::
